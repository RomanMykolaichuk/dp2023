Загальний опис інформаційної системи(лабораторні роботи №№1-3)

Забезпечення CRUD операцій для інформаційного сайту на задану тематику (lab-themes.html).

Front-end: Angular
Back-end: Java (Maven, Servlet, Json, OOP, Collections), Tomcat, FS

На сторінці відображаються декілька об'єктів за встановленою темою (назава картинка коротка характеристика). 
Забезпечується виконання CRUD операцій із збереженням інформації у файлі.

Етапи створення:
Система створюється у три етапи, кожен з яких виконується у вигляді лабораторної роботи.

Завдання на лабораторну роботу №3.
Лабораторна 3. CRUD операції. 
Інтерфейс користувача (Angular) з виконанням CRUD операцій 
через Servlet. 


1. На основі гілки git репозиторію 
   з результатами попередньої лабораторної роботи lab2 створити гілку lab3.
2. В межах зазначеного проекту удосконалити back-end частину,
   за рахунок забезпечення виконання усіх CRUD операцій з використанням Collection Framework
   та збереження інформації у файл.
3. В межах зазначеного проекту завершити розробку front-end частини.
4. Забезпечити сумісну роботу розроблених у п.п. 2, 3 частин системи.
5. Результати викласти у репозиторій GitHub у гілці lab3.
6. Підготуватися до захисту проекту
