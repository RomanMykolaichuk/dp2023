Загальний опис інформаційної системи(лабораторні роботи №№4-9, а також
груповий проект(+ забезпечити декілька пов'язаних таблиць у базі даних))

Забезпечення роботи інформаційного сайту на задану (обрану) тематику.

Front-end: Angular
Back-end: Java (Maven, Spring, REST), PostgreSQL.

На сторінці відображаються декілька об'єктів за темою (назва, картинка, коротка характеристика). 
Забезпечується виконання CRUD операцій із збереженням інформації у базі даних.

Етапи створення:
Система створюється у декілька етапів, кожен з яких виконується у вигляді лабораторної роботи.


Завдання на лабораторну роботу №6.
Лабораторна 6. Створення REST сервісу з використанням анотації @RestController. 

1. В git репозиторії створити гілку lab6.
2. Провести необхідне удосконалення front-end частини, створеної у лаб 4.
3. Розробити REST сервіс з використанням анотації @RestController:
      забезпити виконання CRUD операцій для збереження інформації в базі даних Postgres;
      для роботи з базою даних використати Spring Boot JPA;
      для створення Entity класу використати Lombok.  
4. Забезпечити сумісну роботу розроблених у п.п. 2, 3 частин системи.
5. Результати викласти у репозиторій GitHub у гілці lab6.
6. Підготуватися до захисту проекту.
