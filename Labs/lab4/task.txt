Загальний опис інформаційної системи(лабораторні роботи №№4-9)

Забезпечення роботи інформаційного сайту на задану (обрану) тематику.

Front-end: Angular
Back-end: Java (Maven, Spring, REST), PostgreSQL.

На сторінці відображаються декілька об'єктів за темою (назва, картинка, коротка характеристика). 
Забезпечується виконання CRUD операцій із збереженням інформації у базі даних.

Етапи створення:
Система створюється у декілька етапів, кожен з яких виконується у вигляді лабораторної роботи.


Завдання на лабораторну роботу №4.
Лабораторна 4. Пілотний зразок індивідуального  проекту. 
Робота з базою даних з використанням технологій JDBC, JPA. 

1. В git репозиторії створити гілку lab4.
2. Розробити демо (пілотну) версію індивідуального проекту 
для подальшого виконання на лабораторних роботах 4-9.
3. У front-end частині відпрацювати мінімально функціональний інтерфейс 
для ознайомлення з можливостями програми.
4. В back-end частині виконати:
      створити базу даних (PostgreSQL) для забезпечення потреб проекту;
      відображення інформації на сторінці сайту за рахунок використання технології JDBC (лекція 4);
      забезпечення виконання CRUD операцій для збереження інформації в базі даних з використанням Hibernate (лекція 5).  
5. Забезпечити сумісну роботу розроблених у п.п. 2, 3 частин системи.
6. Результати викласти у репозиторій GitHub у гілці lab4.
7. Підготуватися до захисту проекту.
