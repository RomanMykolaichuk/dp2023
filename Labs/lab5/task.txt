Загальний опис інформаційної системи(лабораторні роботи №№4-9)

Забезпечення роботи інформаційного сайту на задану (обрану) тематику.

Front-end: Angular
Back-end: Java (Maven, Spring, REST), PostgreSQL.

На сторінці відображаються декілька об'єктів за темою (назва, картинка, коротка характеристика). 
Забезпечується виконання CRUD операцій із збереженням інформації у базі даних.

Етапи створення:
Система створюється у декілька етапів, кожен з яких виконується у вигляді лабораторної роботи.


Завдання на лабораторну роботу №5.
Лабораторна 5. Створення програми для адміністрування даних
з використанням патерну MVC. 

1. В git репозиторії створити гілку lab5.
2. Розробити програму адміністрування даних (заповнення та редагування бази даних).
3. У front-end частині забезпечити інтерфейс користувача для виконання 
CRUD операцій в базі даних з використанням Spring MVC, Thymeleaf.
4. В back-end частині виконати (на основі технологій Spring MVC, Spring JPA):
      відображення інформації на веб-сторінці;
      забезпечення виконання CRUD операцій для збереження інформації в базі даних.  
5. Забезпечити сумісну роботу розроблених у п.п. 3, 4 частин програми.
6. Результати викласти у репозиторій GitHub у гілці lab5.
7. Підготуватися до захисту проекту.
