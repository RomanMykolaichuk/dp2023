Загальний опис інформаційної системи(лабораторні роботи №№1-3)

Забезпечення CRUD операцій для інформаційного сайту на задану тематику (lab-themes.html).

Front-end: Angular
Back-end: Java (Maven, Servlet, Json, OOP, Collections), Tomcat, FS

На сторінці відображаються декілька об'єктів за встановленою темою (назава картинка коротка характеристика). 
Забезпечується виконання CRUD операцій із збереженням інформації у файлі.

Етапи створення:
Система створюється у три етапи, кожен з яких виконується у вигляді лабораторної роботи.

Завдання на лабораторну роботу №2.
Лабораторна 2. Інтерфейси та робота з файлами. 
Інтерфейс користувача (Angular) з отриманням та редагуванням інформації 
у форматі JSON через Servlet (doGet, doPut). 
Приклад відображення та редагування одного об'єкту.
Збереження інформації у файлі.

1. На основі гілки git репозиторію 
   з результатами попередньої лабораторної роботи lab1 створити гілку lab2.
2. Створити IntelliJ IDEA  maven project для розробки зазначеної інформаційної системи.
3. В межах зазначеного проекту удосконалити back-end частину,
   за рахунок використання інтерфейсів, збереження інформації у файлі 
   та забезпечення редагування інформації (метод doPut).
4. В межах зазначеного проекту удосконалити front-end частину, 
   за рахунок додавання форми редагування об'єкту 
   та сторінки з відображеням UML діаграми класів проекту
   (згідно з відео lection2 part 1,2 
   https://www.youtube.com/watch?v=laEi-Kp0FK0&list=PLmt9GOKi3Vd4tKijsAa6cE22ZE3-3doqZ).
5. Інтегрувати розроблені у п.п. 3, 4 частини системи.
6. Результати викласти у репозиторій Github у гілці lab2.
7. Підготуватися до захисту проекту
